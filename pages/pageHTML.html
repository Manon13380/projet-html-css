Langage HTML

Qu'est ce que le HTML ?

Le HTML repose sur une syntaxe simple et rigide, dont l'unité de base est la balise. Les balises vont toujours par deux,
une pour ouvrir l'élément, l'autre pour le fermer.
Elles se construisent toujours de la même façon :

  • un chevron "<" pour ouvrir la balise
  • Un attribut: par exemple "p" pour paragraphe, "h1" pour le titre principal
  • un chevron ">" pour fermer la balise
  • Un contenu: le texte à mettre
  • Pour fermer la balise : remettre un chevron ouvrant "<", un slash "/" , l'attribut, et un chevron fermant ">"

Les balises

(Passer la souris sur le tableau pour faire apparaître des exemples de balises)

  • a : création d'une ancre pour liens hypertextes.
  • br : Saut de ligne.
  • form : Insertion d'un formulaire.
  • h1 à h6 : Titres de niveau 1 à 6.
  • img : insertion d'une image.
  • li : Elément d'une liste.
  • p : Début d'un paragraphe.
