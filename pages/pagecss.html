Langage CSS

Qu'est ce que le CSS ?

Le CSS est le code utilisé pour mettre en forme une page web.
C'est un langage de feuille de style, c'est-à-dire qu'il permet d'appliquer des styles
sur différents éléments sélectionnés dans un document HTML.

Pour que la mise en forme CSS s'applique sur la page HTML,

Anatomie d'une règle CSS

Sélecteur

C'est le nom de l'élément HTML situé au début de l'ensemble de règles. Il permet de sélectionner les éléments sur lesquels appliquer le style

Déclaration

C'est une règle simple comme color: red; qui détermine les propriétés de l'élément que l'on veut mettre en forme.

Propriétés

Les différentes façons dont on peut mettre en forme un élément HTML (dans ce cas, color est une propriété des éléments p).

Valeur de la propriété

Celle-ci permet de choisir une mise en forme parmi d'autres pour une propriété donnée (par exemple, il y a d'autres couleurs que red pour la propriété color).

Les propriétés

(Passer la souris sur le tableau pour faire apparaître des exemples de balises)

  • Background : Propriétés de l'arrière-plan.
  • Background-color : Couleur de l'arrière-plan.
  • Border : Caractéristiques des bordures.
  • Color : Couleur de l'avant plan (texte, par exemple).
  • Font-family : Police de caractères.
  • Height : Hauteur de l'élément.
  • Width : Largeur de l'élément.
