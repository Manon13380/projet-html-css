Qu'est ce que le déveleppement Web ?

Le développement Web est le codage ou la programmation qui permet de faire fonctionner un site Web, selon les exigences du propriétaire.
Il existe différents langages pour qu'un site internet prenne vie.

Les différents type de langages

HTML : Soit le squelette de notre site

CSS : Soit la peau de notre site

JavaScript : Soit les muscles de notre site

PHP : Soit le cerveau de notre site
